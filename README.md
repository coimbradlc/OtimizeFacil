# OtimizeFácil

App simples que fiz pra ajudar idosos a limpar o celular.

## O que ele faz
- Simula limpeza (cache, bateria, arquivos)
- Guarda a data/hora da última limpeza para você não esquecer
- Mostra **7 dicas completas** para manter o celular rápido
- Abre antivírus grátis (Malwarebytes)

## Pra quem é
Pro pai, avó, tias e vizinhos que reclamam que o celular está lento.

## Como rodar (passo a passo)

1. Abre o Prompt de Comando (Windows) ou Terminal (Mac)
2. Digita linha por linha:

git clone https://github.com/coimbradlc/OtimizeFacil.git
cd otimizefacil
npm install
npx expo start

3. Baixa o **Expo Go** no celular (grátis na Play Store)
4. Escaneia o QR code que aparece
5. Pronto! O app roda no seu celular
